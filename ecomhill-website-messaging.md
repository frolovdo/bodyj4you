# eCom Hill — Tone & Messaging

**What eCom Hill is:** an **expert-for-hire agency** for Amazon & Walmart sellers — not an
"Amazon help" product. You hire experts by the hour, or retain a month. That model is correct
and stays.

**The whole principle (no tricks):**
- **Be plain.** No marketing embellishment, no manufactured "outcomes that sell," no spin.
- **Release the pattern, then the outcome.** Each item = *the problem* → *the fix*. Nothing else.
- **The outcome is obvious, and it's not free.** Being upfront that it costs — per hour, or a
  monthly retainer — *is* the honest sell.
- **A good page is a page with no issue.** That's the entire promise.

Keep the personality (the apology-dog concept, the calm voice, Denis + Neo) — that's character,
not a sales trick. Everything below just makes the copy plainer and removes the selling tricks.

---

## Pricing — KEEP as-is

"WHO YOU'RE HIRING, BY THE HOUR" ($140 Senior PM / $110 PM / $80 Specialist) and the monthly
tiers (Under $1K / $1–2K / $2–6K) **stay**. This is an expert-for-hire agency; hiring by the
hour or retaining a month is exactly the offer. Keep:

- The hourly rates by role.
- "Everything is billed by the hour, on demand — no contracts, no minimums, no retainer
  padding. We tell you the hour estimate before we start."
- "Committing $4,000+/month (or a large one-time project) drops the rate."

The honesty *is* the pitch. Don't dress it up.

---

## "What we fix" — pattern → outcome, nothing extra

The only change: drop the `HIGH / MEDIUM / LOW` severity pills and let each line be **the
problem, then the fix.** No severity labels, no turnaround badges, no embellishment.

Format for every card: **`Problem (the pattern) → Fixed (the outcome).`**

| Pattern | Outcome |
|---|---|
| Suppressed / deactivated ASIN | Back online. |
| ASIN locked out of PPC | Ad eligibility restored. |
| Account health violation | Standing repaired. |
| New product upload stuck | Upload goes through; launch ships. |
| Flat-file / code error | Error code translated into the actual fix. |
| Brand name change | Live on every listing, no compliance flag. |
| Manufacturer name wrong | Corrected across the catalog. |
| UPC / GS1 mismatch | Mapped to valid GS1 data. |
| No barcode available | GTIN exemption secured or repaired. |
| Variations won't stick | Parentage built so it holds. |
| Parent-child broken | Diagnosed and re-attached. |
| Buried in search | Titles, bullets, backend terms tuned for rank. |
| Wrong category / node | Moved to the node buyers actually use. |

Walmart: same format, same pattern → outcome.

That's it for this section. The outcome is obvious; we don't oversell it.
